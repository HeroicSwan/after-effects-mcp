# Reference Data

Place local media in the matching folder:

- `incoming/`: downloads that have not been categorized
- `finished-good/`: examples to imitate
- `finished-bad/`: examples containing decisions to avoid
- `raw/`: unedited source footage
- `raw-final-pairs/`: matched raw and finished stories
- `narration/`: voice-over and external audio

Media files are intentionally excluded from Git.
